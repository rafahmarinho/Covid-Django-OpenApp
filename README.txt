This project was developed with Django, Postgresql and REST API. The database is hosted on an AWS server. To run the project on your machine, run the steps below:

** Install:

- Python 
- VSCode
- PGAdmin (only if you need something in database)
- Postman (recommended, for requests in localhost)



** Install modules Django projects:

Open CMD on main src:
	- pip install django
	- pip install -r requirements.txt


Run with this command:python manage.py runserver
